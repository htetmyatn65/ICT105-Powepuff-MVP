## Room Booking System Prototype
## Project Overview

This project is a prototype of a Room Booking System developed for a university IT venture project. The system aims to simplify the process of reserving rooms by allowing users to check availability and make bookings online.

## Objective

To provide a convenient and efficient platform for students and staff to reserve rooms without relying on manual booking procedures.

View available rooms <br>
Book a room <br>
Cancel a booking <br>
Check booking schedules <br>
